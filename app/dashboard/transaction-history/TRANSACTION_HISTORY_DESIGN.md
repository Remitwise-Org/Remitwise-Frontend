## Overview
Designed transaction history UI with search, filters, and table density.

## Features
- Search input
- Filters (date, status, type)
- Table density toggle
- Responsive layout (mobile, tablet, desktop)
- Transaction Details page

## States
- Loading
- Empty
- Error
- No results
